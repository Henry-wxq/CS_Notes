# Heading Elements of HTML
## Basics
1. Tags
   1. Opening Tag: <>
   2. Closing Tag: </>
2. HTML Elements: content between opening tag and closing tag and the tags

## Heading
1. From h1 - h7

```
<h1>Heading 1</h1>
```

## Rules
1. Try not have more than one h1 tag.
2. Don't skip a level when creating a head level.
